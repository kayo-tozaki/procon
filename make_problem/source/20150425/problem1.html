Question

学内プログラミングコンテストへようこそ！
競技プログラミングをみんなでやっていこう！

AをBを割った余りを求めて欲しいのです.
C言語において余りを求める計算には"%"演算子が用いれらます.
よってC言語でAをBで割った余りを求めたいときには以下の様に書きます.

A % B

具体例を挙げると
5 % 3 = 2
6 % 3 = 0
10 % 11 = 10
これにより偶奇判定を"%2"よって判断することはしばしばあります.

これで%演算子の使い方が分かりました.それではAをBで割った余りを出力してください.

Input

入力は以下のような形式で与えられます.

A B

Constraints

A:0≦A≦100
B:0≦B≦100

Output

r

r:AをBで割った余りを出力してください. 出力の最後には改行を出力してください.

Sample

Sample Input 1

5 3

Sample Output 1

2

これは上記の例です

Sample Input 2

2 2

Sample Output 2

0

Sample Input 3

99 27

Sample Output 3

18
Do your best.

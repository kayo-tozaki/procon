Time Limit : 2 sec, Memory Limit : 65536 KB

Question

素数が大好きなMくんは，素数を見るだけで幸せな気分になります．つまりある数aが与えられたとき,できるだけ多くの素数の和に変換することで,よりMくんを幸せにすることができます.

Mくんを可能な限り幸せにするために,ある数aできるだけ多くの素数を使った和の式にしてください．

式のルール

  • 使用する素数を小さい順に並べる
  • 素数 半角スペース 半角プラス記号 半角スペース 素数の表示をすること.
  • 式の最後にのみ改行(途中の不要な改行はWAとされます.)

(例) P1 + P2 + P3 + ... + Pn

Input

入力は以下のような形式で与えられます.

a

Constraints

a:2≦a≦1000000の整数

Output

s

s:式のルールを満たすような文字列.

Sample

Sample Input 1

3

Sample Output 1

3

これは上記の例です

Sample Input 2

4

Sample Output 2

2 + 2

Sample Input 3

6

Sample Output 3

2 + 2 + 2
Do your best.

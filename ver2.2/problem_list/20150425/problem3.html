一番強いのは誰だ！？

Time Limit : 2 sec, Memory Limit : 65536 KB

Question

プログラミングコンテストではしばしばレートと呼ばれる格付けのようなものがあります.
普段の競技ではレートが下の人が上の人に勝つことはそのコンテストの頑張りしだいです.しかし今から行われるコンテストではレートは絶対的なものとして扱われます.レートがAの人はレートがA未満の人よりも必ず強いとします.また与えられるレートに同一の値は無いことが保証されています.

今のコンテスト人数Nが与えられます.参加者には0〜n-1までの番号が割り振られ,i番目の人のレートがaiとします.このN人の中で一番強い人の番号を出力してください.またここでいう"強い"とは上で説明した意味だけを持ち,それ以外を表しません.

Input

入力は以下のような形式で与えられます.

N
a0 a1 ... an-1

Constraints

N:0≦N≦100
ai:0≦ai≦100

Output

i

i:一番強い人の番号を出力してください.また出力の最後に改行を出力してください.

Sample

Sample Input 1

5
1 2 3 4 5

Sample Output 1

4

これは上記の例です

Sample Input 2

3
3 2 1

Sample Output 2

0
Do your best.

8乗6倍バズーカ

Time Limit : 2 sec, Memory Limit : 65536 KB

Question

勇者を目指して日々特訓中のMくんは，必殺技の「８乗６倍バズーカ」を習得中です．
この必殺技の威力は，その時の精神力に比例し，関数で表すことできます．
F(精神力) = (精神力^8 + 6*精神力 ) ÷ 2
と定義されます.

Mくんは，精神力を十分に高めることができませんが，少しずつ「８乗６倍バズーカ」を打てるようになってきました．
精神力が与えられるので「８乗６倍バズーカ」の威力を求めてください.

Input

入力は以下のような形式で与えられます.

a

Constraints

a:0≦a≦50の整数

Output

n

n:nは小数点以下1桁まで正確に出力すること.

Sample

Sample Input 1

4

Sample Output 1

32780

Sample Input 2

44

Sample Output 2

7024111812740
